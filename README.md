🧾 Sistema de Inventarios Fullstack

Aplicación web fullstack desarrollada con Angular y Spring Boot para la gestión de productos en un sistema de inventarios.

---

🚀 Tecnologías utilizadas

* ⚡ Angular (Frontend)
* ☕ Spring Boot (Backend)
* 🗄️ MySQL (Base de datos)
* 🔗 API REST

---

📦 Funcionalidades

* ✔ Listar productos
* ✔ Conexión frontend-backend
* ✔ Consumo de API REST
* ✔ Visualización dinámica de datos

⚙️ Instalación

🔹 Backend (Spring Boot)

```bash
cd backend
./mvnw spring-boot:run
```

🔹 Frontend (Angular)

```bash
cd frontend
ng serve
```

---

🌐 Acceso

* Frontend: http://localhost:4200
* Backend: http://localhost:8080
